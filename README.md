# Expense Tracker

A modern and user-friendly Expense Tracker application that helps users manage their daily expenses, monitor spending habits, and gain insights into their financial activities.   

🔗 Live Demo: [https://reeya0409.github.io/expense-tracker/](https://reeya0409.github.io/expense-tracker/)   

---

## 📌 Overview

Managing personal finances can be challenging when expenses are not tracked properly. This Expense Tracker provides an easy way to record income and expenses, categorize transactions, and visualize spending patterns through interactive dashboards and reports.

The application is designed to help users:

* Track daily expenses
* Monitor spending habits
* Manage personal finances efficiently
* Maintain better budgeting discipline

---

## 🚀 Features

* Add new expenses
* Delete expenses
* View total expenses
* Visualize expenses using charts and graphs
* Responsive design for mobile and desktop
* Data persistence using local storage
* Simple and user-friendly interface
